# 网易云音乐车机版 - 状态栏控制中心模块

[![LSPosed](https://img.shields.io/badge/LSPosed-API%20101-brightgreen)](https://github.com/LSPosed/LSPosed)

## 项目简介

本项目是一个基于 **LSPosed API 101** 的 Xposed 模块，旨在为**不支持状态栏控制条**的网易云音乐车机版，添加一个状态栏控制中心模块。

项目目前处于 **极早期阶段**，功能尚在完善中，欢迎尝鲜与反馈。

## 功能特性

- 为网易云音乐车机版添加状态栏控制条
- 支持基础的播放控制（播放/暂停、上一曲、下一曲）
- 轻量化实现，适配车机环境

## 使用要求

- 已 Root 的 Android 设备
- 已安装 [LSPosed](https://github.com/LSPosed/LSPosed) 框架
- 网易云音乐车机版。

## 安装与使用

1. 在 LSPosed 管理器中启用本模块
2. 勾选网易云音乐车机版应用
3. 重启网易云音乐应用
4. 播放音乐时，状态栏将显示控制中心模块

## 兼容性
