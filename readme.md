# NCM Car Notify

**包名：** `io.github.lostmymind.ncm.car.notify`

为网易云音乐车机版补全通知中心控制栏功能的 Xposed 模块。

---

## 📌 项目简介

网易云音乐车机版相比标准版，去除了大量广告与冗余功能，整体更加轻量、流畅且功耗更低，非常适合车机或低性能设备使用。

但与此同时，车机版也缺失了一些在手机端较为实用的功能，例如**通知中心媒体控制栏**。

本模块的目标就是：

> ✅ 为网易云音乐车机版补全通知中心控制条功能  
> ✅ 提供更接近标准版的媒体控制体验

---

## ❓ 为什么选择车机版？

相比标准版，网易云音乐车机版具有以下优势：

- 🚀 更流畅（去除大量非核心功能）
- 🔋 更低功耗（适合长时间后台播放）
- 🧹 更少广告干扰

但缺点也很明显：

- ❌ 缺少通知栏控制
- ❌ 部分交互功能缺失

👉 本模块即用于弥补这些不足。

---

## ⚙️ 关于 API101 与 API93 双版本

本项目提供两个版本：

| 版本 | 基于 | 特点 |
|------|------|------|
| API101 | LibXposed API101 | 新架构，未来趋势 |
| API93  | Xposed API93     | 兼容性强，支持广泛 |

### 🔍 选择建议：

- **使用 LSPosed 2.0+（新环境）** → 推荐 API101
- **使用旧框架 / 免 Root 框架** → 推荐 API93

### ⚠️ 重要说明：

- LSPosed 正在逐步淘汰旧 API
- 已确认 API93 在未来（API102）可能出现行为变更甚至失效
- 但目前：
    - API93 仍然拥有**更好的兼容性**
    - API101 **暂无免 Root 框架支持**

---

## 🚀 使用方法

1. 根据你的 Xposed / LSPosed 框架版本下载对应 APK：
    - API101 或 API93
2. 安装模块
3. 在作用域中勾选：
    - **网易云音乐车机版**
4. 重启应用（必要时重启系统）
